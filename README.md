# Projetos dos cursos da DIO #

Este repositório contém todos os projetos elaborados para os *bootcamps* da [DIO](https://dio.me).

## Projetos (2024) ##

* **Azure Docker Microserviços:** demonstração de microserviços usando Docker, disponível na pasta [2024-azure-docker-microservices](2024-azure-docker-microservices/).

* **Azure Functions CPF Validator:** validação de CPF usando Azure Functions, disponível na pasta [2024-azure-function-cpf](2024-azure-function-cpf/).

* **Azure Kubernetes Pipeline:** demonstração de pipeline Kubernet, disponível na pasta [2024-kubernetes-pipeline](2024-kubernetes-pipeline/).

* **Azure WebApps API:** demonstração de Azure Web Apps, disponível na pasta [2024-azure-web-app](2024-azure-web-app/)

* **Validação de cartão de crédito usando Azure Document Intelligence:** exercício de Azure disponível na pasta [2024-azure-credit-card-evaluator](2024-azure-credit-card-evaluator/).

* **Tradutor de textos usando Azure Translator e Azure OpenAI:** exercício de Azure e OpenAI disponível na pasta [2024-azure-translator](2024-azure-translator/).

* **Java Spring Conta Bancária:** classe para simular um sistema de conta bancária em Java, disponível na pasta [2024-conta-bancaria](2024-conta-bancaria/).

* **Java Spring Controle de Fluxo:** classe para demonstrar controle de fluxo em Java, disponível na pasta [2024-controle-fluxo](2024-controle-fluxo/).

* **Java Banco Digital:** exercício de orientação a objetos em Java, disponível na pasta [2024-java-banco-digital](2024-java-banco-digital/).

* **Java Bootcamp:** estrutura de dados para gerenciamento de bootcamp em Java, disponível na pasta [2024-java-bootcamp](2024-java-bootcamp/).

* **Java API REST:** demonstração de API RESTful em Java e Spring, disponível na pasta [2024-java-spring-api-rest](2024-java-spring-api-rest/).

* **Java Design Patterns:** demonstração de *design patterns* em Java e Spring, disponível na pasta [2024-java-spring-patterns](2024-java-spring-patterns/).

* **Jogo Detona Ralph em Javascript:** jogo Detona Ralph em Javascript, disponível na pasta [2024-javascript-detona-ralph](2024-javascript-detona-ralph/).

* **Javascript Classificação de Herói:** classe para demonstrar programação básica em Javascript, disponível na pasta [2024-javascript-hero](2024-javascript-hero/).

* **Javascript landing-page:** land page escrita em HTML/Javascript, disponível na pasta [2024-javascript-landingpage](2024-javascript-landingpage/).

* **Javascript Memory Game:** jogo da memória desenvolvido em HTML/Javascript, disponível na pasta [2024-javascript-memory-game](2024-javascript-memory-game/).

* **Javascript Piano:** simulador de piano em Javascript, disponível na pasta [2024-javascript-piano](2024-javascript-piano/).

* **Javascript Pokedex:** demonstração de CSS e Javascript, disponível na pasta [2024-javascript-pokedex](2024-javascript-pokedex/).

* **Javascript Yo-Gi-Oh:** jogo de Yo-Gi-Oh desenvolvido em Javascript, disponível na pasta [2024-javascript-yo-gi-oh](2024-javascript-yo-gi-oh/).

* **Netflix using Azure Functions:** servidor de vídeos usando Azure Functions e Cosmos DB, disponível na pasta [2024-azure-netflix](2024-azure-netflix/).

* **Nexa AWS Personal Trainer:** personal trainer escrito em AWS Bedrock para o projeto Nexa, disponível na pasta [2024-nexa-aws-personal-trainer](2024-nexa-aws-personal-trainer/).

* **Nexa AWS Step Functions:** modelo desenvolvido no AWS Step Functions, disponível na pasta [2024-nexa-aws-step-functions](2024-nexa-aws-step-functions).

* **DIO iPhone UML:** exercício de UML (*Unified Modelling Language*), disponível na pasta [2024-uml-iphone](2024-uml-iphone/).
